How to Use GitAuto

Connect to GitAuto

To connect to GitAuto, use the shortcut:

Ctrl + Shift + P

Type "Connect to GitAuto" and follow these steps:

  • Enter your email when prompted.
  • Enter your password when prompted.
  • If successful, see: "Welcome, [Your Name]!"

Illustration of connecting to GitAuto in VSCode

Generate Commit Message

To generate commit messages:

Ctrl + Shift + P

Type "Generate commit message" to:

  • Automatically create messages for each working tree change.
  • Add each staged change with its own message.

Illustration of generating commit messages

Push Latest Changes

To push changes to your repository:

Ctrl + Shift + P

Type "Push latest changes" to:

  • Generate any remaining commit messages for unstaged changes.
  • Push all commits to your remote repository.

Illustration of pushing changes to the repository
